# Hobby Projects That Actually Level Up Your Career

Good side projects prove execution, not just ideas.

## Strong project ingredients

- Real auth flow
- Clear API contracts
- Database migrations
- Deployment + monitoring

## Publish proof

- Architecture decisions
- Tradeoff notes
- Short postmortem

Evidence beats claims in interviews and collaborations.
