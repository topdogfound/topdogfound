# AI-Assisted Coding Without Losing Skill

AI can accelerate execution, but judgment still determines outcomes.

## Good AI use cases

- Boilerplate
- Refactor drafts
- Test scaffolding
- Documentation first drafts

## Human-led decisions

- Architecture
- Security
- Domain tradeoffs
- Incident response

If you cannot explain the generated change, do not merge it.
