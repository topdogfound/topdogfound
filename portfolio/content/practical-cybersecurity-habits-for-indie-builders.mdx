# Practical Cybersecurity Habits for Indie Builders

Security should be routine, not a panic response.

## Non-negotiables

- MFA on critical services
- Password manager with unique credentials
- Secret scanning in CI
- Dependency update checks

## App-level basics

- Validate input server-side
- Use least-privilege roles
- Log security-relevant events

Start with basics done consistently.
