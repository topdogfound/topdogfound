# Open Source Contribution Playbook

You do not need massive pull requests to contribute meaningfully.

## High-leverage contributions

- Documentation improvements
- Test coverage additions
- Bug reproductions with minimal cases
- Small reliability refactors

Consistency builds trust with maintainers.
