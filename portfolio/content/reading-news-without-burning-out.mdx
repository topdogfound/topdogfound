# Reading News Without Burning Out

Staying informed should improve decisions, not destroy concentration.

## Better media hygiene

- Fixed reading windows
- Fewer sources, higher quality
- More analysis, less outrage content

## Useful filter

Ask: "Does this change a decision I need to make this week?"

If not, acknowledge and move on.
