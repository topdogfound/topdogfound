# Deep Work in an Interruption Economy

Deep work is a design problem, not a motivation problem.

## Daily structure

- 90-minute focus block first
- Meetings clustered later
- Admin in a fixed time slot

## Team rules

- Use async docs first
- Set response-time expectations
- Reduce ad-hoc call culture

Protected focus time compounds output quality.
